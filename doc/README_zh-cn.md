和弦（Kord）

和弦是一个社区驱动的，基于按键语义定义文件的精确按键冲突识别项目。它的目标是为用户提供一个准确的按键冲突识别工具，以帮助他们更好地理解和管理他们的按键设置。

在中大型Minercraft整合包中，按键设置条目可能会非常庞大，而原版和NeoForge的按键冲突检测功能非常孱弱，且非常容易产生误判和错判，导致用户无法准确地识别和解决按键冲突问题。通过引入精确的按键语义定义，和弦可以更加智能的识别按键冲突，减少用户在按键设置中的心智负担。由于各类Mod的按键功能纷繁复杂，和弦的按键语义定义文件也将由社区成员共同贡献，以期待能够覆盖更多的按键功能和使用场景。

和弦目前和计划中的主要功能包括：
1. 精确的按键冲突识别：通过引入按键语义定义文件，和弦能够更准确地识别按键冲突
2. 社区驱动的按键语义定义库：和弦将建立一个社区驱动的按键语义定义库，用户可以创建和共享他们的按键语义定义，以帮助其他用户更好地识别按键冲突
3. 用户自定义的按键冲突数据：在和弦的算法不能覆盖的情况下，用户可以手动添加按键冲突数据，以帮助和弦更好地识别按键冲突
4. 一个全新的按键设置界面：和弦将提供一个全新的按键设置界面，用户可以在其中管理他们的按键设置，并且能够更清晰地看到按键冲突信息
5. 按键冲突结果的编译与缓存：对于大型整合包，和弦将提供按键冲突结果的编译与缓存功能，以提高按键设置界面的响应速度和用户体验
etc...
